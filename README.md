## Мои лучшие проекты

### 1. HR-ассистент по отбору кандидатов
**Стек:** `n8n`, `Gmail API`, `Google Sheets`, `AI (парсинг резюме)`

Автоматизированный помощник, который мониторит отклики на вакансии и первично оценивает кандидатов.
*   Мониторинг почты Gmail на предмет новых откликов с резюме.
*   Автоматическое извлечение ключевых данных из резюме по заданному формату.
*   Сравнение опыта кандидата со списком требований из Google Sheets.
*   Присвоение рейтинга кандидату и формирование списка лучших.

### 2. Telegram-бот для поиска горящих туров в Турцию
**Стек:** `Python`, `Парсинг`, `База данных`

Оперативный помощник для поиска лучших предложений по отдыху.
*   Автоматический сбор горящих предложений с заданных сайтов.
*   Извлечение заголовка тура, описания, фото отеля и прямой ссылки.
*   Сохранение информации в базу данных.
*   Отправка пользователю по команде `/promo` до 10 самых актуальных спецпредложений с номерами, заголовками и ссылками.

### 3. Telegram-бот для танцевальной студии
**Стек:** `Salebot`, `Интеграции`, `CRM`

Полноценный инструмент для записи клиентов и автоматизации коммуникаций студии.
*   Удобная запись на занятия через кнопки в разделах «Расписание» и «Стоимость».
*   Сбор контактных данных (Имя, телефон, email).
*   Автоматическая отправка подтверждения записи пользователю.
*   Напоминание клиенту о занятии за 24 часа с кнопкой подтверждения.
*   Сохранение всех подтверждений в Google Sheets для учета.
